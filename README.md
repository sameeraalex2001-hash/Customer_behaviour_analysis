# Customer_behaviour_analysis

## Project Overview
This project is an end-to-end data analytics project that analyzes consumer purchasing behavior using Python, SQL, and Power BI. The objective is to extract meaningful insights from customer transaction data and support data-driven decision making.

## Objectives
- Understand consumer purchase patterns
- Identify high-value customers
- Analyze spending behavior across locations
- Discover trends that can support marketing strategies

## Tools and Technologies
- Python (pandas, numpy, matplotlib)
- SQL
- Power BI
- Excel

## Project Workflow
1. Data Collection – Imported customer transaction dataset.
2. Data Cleaning – Removed missing values and corrected data formats.
3. Exploratory Data Analysis (EDA) – Analyzed customer behavior, spending patterns, and distribution trends.
4. SQL Analysis – Wrote queries to extract business insights from the dataset.
5. Feature Engineering – Created additional variables to improve analysis.
6. Data Visualization – Built dashboards and visualizations to present insights.

## Key Analysis Performed
- Top customers based on purchase amount
- Customer distribution by location
- Purchase frequency analysis
- Customer segmentation based on spending behavior
- Trend analysis of purchases

## Dashboard Features
The Power BI dashboard includes:
- Total revenue and purchase metrics
- Customer distribution by city
- Top spending customers
- Purchase trends visualization
- Interactive filters for deeper analysis
